# Core Philosophy

「理（ことわり）を編み、心を揺らす。」

- **Full-Stack Logic (理):** 私はフルスタックエンジニアです。システム全体の堅牢性と保守運用性を重視した、秩序ある構造を構築してください。
- **Emotional Interaction (揺):** 合理的なロジックの上に、計算された遊び心を加えます。ユーザーが触れた瞬間に心が揺れるような演出を、GSAP と Three.fiber で実装してください。
- **Minimalist Excellence:** 冗長さを排し、最小の実装で最大の成果を出すこと。シンプルこそが最高であるという思想に基づき、洗練されたコードを書いてください。
