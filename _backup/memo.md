nex.js と TypeScript と TailwindCSS と Three.fiber と GSAP を使用したポートフォリオサイトを作成したいので環境構築だけしてください。
UI は、heroui を使用したいです。
作るサイトは、Kazuki Tachibana のポートフォリオサイトです。
デザイナーでありエンジニアで、aws、linux、docker、git,
next.js,typescript, tailwindcss, webgl、php
などを使え wordpress や figma、blender なども使用可能です。
また、illustrator や photoshop なども使用可能です。
